# Code of Conduct

This project follows a short, plain-English code of conduct:

1. **Act like a citizen.** Treat every contributor — first-timer, regular, maintainer — as someone whose time matters.
2. **Be specific.** Disagree with the idea, not the person. "This rule seems to overfit my use case because X" is welcome; "this is bad" is not.
3. **Default to good faith.** Ask before assuming malice. Most friction is a misunderstanding.
4. **No harassment.** Personal attacks, slurs, threats, or sustained derailment are grounds for a permanent ban with no warning. There is no second chance for harassment.
5. **Public archives stay public.** Issues, PRs, and discussions are part of the project's history. We won't redact them on request unless they contain genuine personal information that shouldn't have been posted.

Enforcement: maintainers can lock, hide, or remove content and ban participants. Decisions are final and discussed in the open in `_polis/chronicle.md` of this repo when warranted.

To report a violation privately, email **yehuda.moshe24@gmail.com**.

Inspired by but not bound to the [Contributor Covenant](https://www.contributor-covenant.org/). This one is shorter on purpose.
