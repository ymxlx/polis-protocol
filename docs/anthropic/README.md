# Anthropic submission package

This folder contains pre-drafted copy for every Anthropic-controlled channel where Polis Protocol can be submitted or surfaced. **Each file is copy-paste ready** — the goal is to make the human action under 5 minutes per channel.

## Priority order (highest leverage first)

| # | Channel | File | Cost | Realistic outcome |
|---|---------|------|------|--------------------|
| 1 | **clau.de plugin directory** | [`claude-de-submission.md`](claude-de-submission.md) | ~10 min | Listed in Claude Code in-app plugin browser within 2–3 weeks |
| 2 | **platform.claude.com/plugins/submit** (Console) | [`claude-de-submission.md`](claude-de-submission.md) (same fields) | ~10 min | Same as above — submit to both for redundancy |
| 3 | **Anthropic Builders Showcase** | [`builders-showcase-application.md`](builders-showcase-application.md) | ~15 min | Featured slot reaches 100k+ devs; reply in 30–90 days |
| 4 | **Anthropic Discord posts** | [`discord-posts.md`](discord-posts.md) | ~15 min | Reaches Claude Code power users + Anthropic DevRel passively |
| 5 | **DMs to Anthropic DevRel** | [`devrel-dms.md`](devrel-dms.md) | ~10 min | Any reply = signal; retweet/post = huge reach |
| 6 | **@AnthropicAI tag tweet** | [`launch-tweet-mention.md`](launch-tweet-mention.md) | ~2 min | Surfaces to their social team; ~5% retweet rate |

**Run all six in one 60-minute block.** They compound — a Builders Showcase reply name-drops to DevRel; a Discord post visible to DevRel makes the DM warmer.

## Channels investigated and skipped

- **`anthropics/skills` (136k★)** — accepts PRs in theory, but zero community-submitted skills have actually merged. The 30+ open community PRs sit indefinitely. Don't waste a submission here; the signal-to-noise destroys the maintainer-trust budget.
- **`anthropics/claude-cookbooks`** — single-vendor Claude API examples only. Polis is explicitly multi-vendor, so this is an architectural mismatch, not a discoverability question.
- **`anthropics/claude-plugins-official`** — already auto-rejected ([PR #1893](https://github.com/anthropics/claude-plugins-official/pull/1893)) with explicit redirect to clau.de form. Don't retry.
- **`anthropics/claude-plugins-community`** — read-only mirror of the clau.de form output. Same as #1.

## Files in this folder

- `claude-de-submission.md` — every field of the clau.de form, pre-filled
- `builders-showcase-application.md` — Anthropic Builders Showcase application copy
- `discord-posts.md` — tailored posts for Anthropic Builders Discord + Claude Code Discord
- `devrel-dms.md` — personalized DMs to specific Anthropic DevRel humans
- `launch-tweet-mention.md` — single-tweet copy that tags @AnthropicAI

## Why this matters

Anthropic-controlled channels are the only place Polis Protocol reaches the exact ICP: Claude Code users running multi-vendor agent setups. Every other channel (HN, Reddit, X general, awesome-lists) is wider but less qualified. Of the 17 awesome-list submissions already in flight, only 2 have merged. The Anthropic channels collectively reach ~100x the qualified audience.
