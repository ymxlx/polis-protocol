# Launch materials

Self-contained launch package. All copy is in markdown; paste straight into the venues.

| File | Use |
|---|---|
| [`essay.md`](essay.md) | Long-form launch essay. Publish on your domain Day 2; mirror to Dev.to + Hashnode 48h later. |
| [`show-hn.md`](show-hn.md) | Show HN title, body, and the first comment to post right after submission. |
| [`x-thread.md`](x-thread.md) | 4-tweet thread + LinkedIn variant + reply-tag list. |
| [`community-pitches.md`](community-pitches.md) | Reddit, Discord, and newsletter-pitch copy. Tailored per venue. |
| [`checklist.md`](checklist.md) | Day-by-day 7-day launch checklist. |

Read order: `checklist.md` first to understand the flow, then the other four as you reach each step.

Three rules through all of it: **respond fast, ship visibly, credit by name.**
